# Chip 标签

标签组件允许用户展示信息、进行选择、过滤内容或触发操作。

使用 `import { ISwitch } from "@/components";`

源码 `components/widgets/switch`

## 何时使用

> Chips 通常是小而圆的视觉元素，可以包含文本、图标，甚至动作（如删除）。

* 用于标记事物的属性和维度。

## API

* 通用属性-`Input`

| 参数 | 类型 | 默认值 | 描述 |
| ---------------- | ------------------- | ------------ | ------------ |
| `checked`        | `boolean`           | `-`          | 控制是否选中。|
| `defaultChecked` | `boolean`           | `-`          | 默认是否选中。|
| `isLoading`      | `boolean`           | `-`          | 控制控件是否加载中。|
| `pattren`        | `ControlPattern`    | `editable`   | 控制控件的交互方式。|
| `size`           | `ControlUISize`     | `medium`     | 控制控件的尺寸大小。|
| `status`         | `ControlUIStatus`   | `success`    | 控制控件的校验状态。|
| `render`         | `ISwitchRenders`    | `-`          | 自定义控件UI。|
