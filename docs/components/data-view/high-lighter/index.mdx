## API

* 通用属性-`Input`

| 参数 | 类型 | 默认值 | 描述 |
| ---------------- | -------------------------- | ------------ | ------------ |
| `content`      | `boolean`                  | `-`          | 控制是否呈现关闭按钮。|
| `keyword`      | `boolean`                  | `-`          | 控制是否呈现可点击样式（与是否定义`onClick`无关）。|
| `compare`           | `React.ReactNode`          | `-`          | 控制组件前缀图标。|
| `render`         | `ISwitchRenders`           | `-`          | 自定义控件UI。|
