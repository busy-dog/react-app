# Modal 模态弹窗

模态弹窗向用户通报任务，并且可以包含关键信息、需要决策或涉及多个任务。

使用 `import { IModal } from "@/components";`

源码 `components/widgets/modal`

## 何时使用

* 模态弹窗通常出现在应用程序内容前面，用于提供关键信息或要求做出决定。弹窗出现时会禁用所有应用程序功能，并保留在屏幕上，直到确认、取消或采取所需操作。
