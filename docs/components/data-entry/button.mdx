# Button 按钮

> 按钮允许用户通过单击来执行操作和做出选择，一般用来传达用户可以执行的操作。

使用 `import { IButton } from "@/components";`

源码 `components/widgets/button`

## 何时使用

* 传达用户可以采取的操作，触发相应的业务逻辑。

## API

| 参数 | 类型 | 默认值 | 描述 |
| ------------- | --------------------------------------- | ------------ | ------------ |
| `capsule`     | `boolean`                               | `-`          | 是否呈现胶囊按钮样式。|
| `danger`      | `boolean`                               | `-`          | 是否呈现危险按钮样式。|
| `debounce`    | `boolean` \| `number`                   | `-`          | 是否启用事件防抖。|
| `icon`        | `ReactNode`                             | `-`          | 设置按钮的图标组件。|
| `isFullWidth` | `boolean`                               | `-`          | 设置按钮宽度是否填满父容器。|
| `isLoading`   | `boolean`                               | `-`          | 设置是否呈现加载按钮样式。|
| `size`        | `ControlUISize`                         | `medium`     | 设置按钮尺寸。|
| `variant`     | `'filled'` \| `'bordered'` \| `'text'`  | `bordered`   | 设置按钮类型。|
| `wave`        | `boolean`                               | `-`          | 是否呈现波纹反馈（在filled类型默认呈现）。|
