# Checkbox 多选框

> Checkbox 允许用户从一组中选择一个或多个项目。

使用 `import { ICheckbox } from "@/components";`

源码 `components/widgets/checkbox`

## 何时使用

* 在一组可选项中进行多项选择时；

* 单独使用可以表示两种状态之间的切换，和 switch 类似。区别在于切换 switch 会直接触发状态改变，而 checkbox 一般用于状态标记，需要和提交操作配合；

* Checkboxe vs Radio；

* Checkboxe vs Switch；

### 不确定

复选框输入在表单中只能有两种状态：选中或未选中。它要么提交其值，要么不提交。从视觉上看，复选框可以处于三种状态：选中、未选中或不确定。

## API

| 参数 | 类型 | 默认值 | 描述 |
| ---------------- | -------------------------------------- | ------------ | ------------ |
| `checked`        | `boolean`                              | `-`          | 控制控件当前是否选中。|
| `defaultChecked` | `boolean`                              | `-`          | 控件是否默认选中。|
| `name`           | `string`                               | `-`          | 控制控件 input 元素的 name 属性。|
| `label`          | `React.ReactNode`                      | `-`          | 控制控件文本内容。|
| `id`             | `string`                               | `-`          | 控制控件 input 元素的 id 属性。|
| `indeterminate`  | `boolean`                              | `-`          | 如果为 true ，则组件显示为不确定。由于兼容性问题这里没有控制 input 的 indeterminate属性。|
| `pattren`        | `ControlPattern`                       | `editable`   | 设置控件的交互方式。|
| `size`           | `ControlUISize`                        | `medium`     | 设置控件尺寸。|
| `status`         | `ControlUIStatus`                      | `success`    | 设置控件的校验状态。|
| `value`          | `ControlValue`                         | `-`          | 控制控件 input 元素的值。|
| `wave`           | `boolean`                              | `-`          | 是否呈现波纹反馈。|
| `render`         | `ICheckboxRenders`                     | `true`       | 自定义控件UI。|
| `onChange`       | `ChangeEventHandler<HTMLInputElement>` | `-`          | 状态更改时触发回调。|
