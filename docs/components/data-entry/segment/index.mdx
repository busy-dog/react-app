# Segment 分段控制器

> Segment Control通常用于切换不同的视图，或者在表单中作为单选组件使用。

使用 `import { ISegment } from "@/components";`

源码 `components/widgets/sgment`

## 何时使用

* 在一组互斥的可选项中进行单项选择时。

## API

### Segment

* 通用属性-([IFlex](/components/typesetting/flex.html))；

| 参数 | 类型 | 默认值 | 描述 |
| ---------------- | -------------------- | ------------ | ------------ |
| `defaultValue`   | `React.Key`          | `-`          | 控件默认选中的分段。|
| `disabled`       | `boolean`            | `-`          | 控制控件是否禁用状态。|
| `readOnly`       | `boolean`            | `-`          | 控制控件是否只读状态。|
| `isFullWidth`    | `boolean`            | `-`          | 控件宽度是否填充父元素。|
| `options`        | `ControlOption`      | `-`          | 控件的选项内容。|
| `size`           | `ControlUISize`      | `medium`     | 控制控件的尺寸大小。|
| `value`          | `React.Key`          | `-`          | 控制控件的值。|
| `render`         | `ISegmentRenders`    | `-`          | 自定义控件UI。|
| `onChange`       | `ISegmentChangeFunc` | `-`          | input标签的值。|
