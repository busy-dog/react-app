# Typography 文本

Typography 组件有助于清晰有效地呈现文本内容。

使用 `import { ITypography } from "@/components";`

源码 `/src/components/widgets/typography`

## 何时使用

* 当文本、段落需要内嵌SVG图标时。

## 代码演示

### 基本用法

Typography 组件通过提供一组有限的可供选择的值以及用于更快地构建通用设计的方便道具来帮助保持一致的设计。

Typography 组件围绕其内容，以特定的排版样式和属性显示文本。

### 标题和正文

使用 h1 到 h4 呈现标题。生成的 HTML 元素将与页面结构的语义标题匹配。
