# Divider 分割线

Divider 组件提供了一条细细的、不显眼的线来对元素进行分组，以加强视觉层次结构。

使用 `import { Divider } from "@/components";`

源码 `/src/components/widgets/divider`

## 何时使用

* 作为容器对子元素进行弹性布局
